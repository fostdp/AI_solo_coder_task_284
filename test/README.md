## 🔬 核心物理模型

### 涡格法 (VLM) 简化实现
```javascript
// 升力线斜率 (3D)
clAlpha_3D = (2 * π * AR) / (2 + sqrt(4 + (AR * β / η)² * (1 + tan(Λ)² / β²)))

// 诱导阻力
Cd_i = Cl² / (π * AR * η)

// 总阻力
Cd = Cd_0 + Cd_i + Cd_wave
```

### 纵向静稳定性判据
```
静稳定裕度 (SM) = X_np - X_cg

其中:
- X_np: 中性点位置
- X_cg: 重心位置

稳定条件: SM > 0 且 dCm/dCl < 0
```

### 射程计算模型
```
总射程 = 0.3 * 弹道射程 + 0.7 * 滑翔射程

弹道射程: R_ballistic = V² * sin(2θ) / g
滑翔射程: R_glide = (L/D) * (h + V²/(2g))
```

## 🐛 调试失败测试

如果测试失败，检查以下可能原因:

1. **数值精度问题**: 调整 `assertApprox` 的容差
2. **边界条件问题**: 检查极值处的物理合理性
3. **参数耦合效应**: 多个参数同时变化时的相互作用
4. **模型假设限制**: 简化模型在极端条件下的适用性
