# ✈️ 纸飞机空气动力学设计游戏

一个交互式的纸飞机设计和模拟游戏，玩家可以调整空气动力学参数，进行风洞测试，并试飞挑战最远距离。

## 🎮 功能特性

### 📐 设计参数
- **机翼后掠角** (0° - 60°): 影响高速性能和低速升力
- **翼展** (5cm - 25cm): 影响升力和阻力
- **重心位置** (30% - 70%): 影响飞行稳定性

### 🌀 模拟功能
- **风洞测试**: 可视化气流，计算升阻比和稳定性
- **试飞模拟**: 真实物理模拟，包括重力、升力、阻力、湍流影响
- **实时结果**: 显示升阻比、稳定性评分、飞行距离

### 💾 数据持久化
- 保存设计参数和最佳飞行距离
- 排行榜显示前10名设计
- 完整的设计参数记录

## 🎯 游戏玩法

1. **设计你的纸飞机**
   - 给你的设计起个名字
   - 调整后掠角、翼展和重心位置
   - 观察Canvas中的纸飞机预览

2. **风洞测试**
   - 点击"风洞测试"按钮
   - 观察气流可视化效果
   - 获取升阻比和稳定性评分

3. **开始试飞**
   - 点击"开始试飞"按钮
   - 观看纸飞机飞行动画
   - 查看最终飞行距离

4. **保存设计**
   - 满意的设计可以保存到数据库
   - 点击"刷新"查看排行榜
   - 挑战最远飞行距离记录！

## 🔬 空气动力学模型

### 升阻比计算
- 最优后掠角: 25°
- 翼展增大 → 升力增大，但阻力也增大
- 后掠角增大 → 高速性能提升，但低速升力下降

### 稳定性计算
- 最优重心位置: 45% (机头方向)
- 重心偏离越大，稳定性越差
- 稳定性影响湍流对飞机的影响程度

## 📊 API 接口

### POST `/api/designs`
保存新设计

**请求体:**
```json
{
  "name": "设计名称",
  "sweepAngle": 30,
  "wingSpan": 15,
  "centerOfGravity": 50,
  "liftToDragRatio": 8.5,
  "stability": 7.2,
  "bestDistance": 15.5
}
```

### GET `/api/designs`
获取排行榜（前10名，按距离排序）

### GET `/api/designs/:id`
获取单个设计详情

### DELETE `/api/designs/:id`
删除指定设计

## 💡 设计技巧

1. 追求高**升阻比**可以飞得更远
2. 保持高**稳定性**可以避免坠毁
3. 重心在45%左右通常有最佳表现
4. 后掠角20°-30°是个好的起点
5. 翼展12cm-18cm平衡了升力和阻力

## 🐛 故障排除

### PowerShell执行策略问题
如果遇到npm脚本执行权限问题：
1. 以管理员身份打开PowerShell
2. 运行: `Set-ExecutionPolicy RemoteSigned`
3. 选择"Y"确认

### 端口被占用
修改`server.js`中的`PORT`变量
